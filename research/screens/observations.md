## ✅ Battle.net (нативний клієнт, **залогінено** — Batman / Online)

**Структура екрана HOME:**
- **Топ-ряд-світч ігор** — горизонтальний ряд іконок (Favorites + усі тайтли Blizzard + кнопка `+`). Масштабується на багато ігор краще, ніж довгий лівий список.
- Верхня нав: `HOME / GAMES / SHOP`, дзвіночок-нотифікації, акаунт (Batman, Online), кнопка поп-аут.
- **Великий авто-ротований герой-банер** (Diablo IV «New expansion now available» + CTA `Purchase Now`).
- **Контент-рейки** під банером (Diablo IV / WoW / CoD MW…) — промо й новини per-game.
- **Постійна права соц-панель**: Favorites, Friends зі статусами («Offline for 10 days/years»), доковані «Chats and Groups».

**Сторінка гри (вкладка тайтлу):** ліва міні-нав `Shop / Patch Notes / Forums / Support`; селектор `GAME VERSION` + глобус-регіон; CTA `Install`/`Play` закріплено внизу-ліворуч; контент-рейки `ESPORTS / PATCH NOTES / SHORT STORY`.

**Висновки для WGC:**
- Топ-ряд іконок як перемикач ігор → модель для **масштабованості портфоліо** WG.
- **Play/Install закріплено в передбачуваному місці** per-game → підтримує time-to-play.
- **Новини живуть усередині гри** (patch notes / esports / story рейки) → референс «контент як двигун».
- Соц-панель завжди присутня (engagement), але важка → **питання до PM #3 (соц-шар у скоупі?)**.
- Монетизація: SHOP у нав + «Purchase Now» у банері — присутня, але без модалок-спаму.

## ✅ GOG Galaxy (нативний клієнт, **залогінено** — artqb9, укр. локаль)

**Бібліотека (Ігри у власності):** чиста сітка **вертикальних box-art**, фільтр платформи (`Mac`), `Відбір` (фільтри), `+` (додати гру), пошук, тумблер кастомізації вигляду. Мінімум хрому, контент-forward.

**Чітка IA в лівому сайдбарі (ключове!):**
- `GOG GALAXY` — Підбірки (collections) / Шукати / Нещодавнє
- `КРАМНИЦЯ GOG` — Домівка / Усі ігри / Пропозиції / Бажане / Кошик / Історія / Dreamlist / Patrons
- `ІГРИ` — Ігри у власності / Установлені
- `ДРУЗІ`

**Store («Усі ігри»):** жанрові плитки (Atmospheric/Tactical/Fantasy/Strategy/Classic), багаті фільтри (`Hide all owned`, `Show only discounted`, `wishlist`, `DLCs`), сорт «Bestselling», grid/list, пагінація (209 стор.), бейджі знижок `-95%`, community-poll банер.

**Оверв'ю гри:** стрічка активності (свої + друзів здобутки/«зіграно»), статистика друзів (% проходження, годинник), DLC-офери.

**Висновки для WGC:**
- **Магазин — окрема зона сайдбару, не накладена на бібліотеку** → це GOG-style відповідь на **питання до PM #1**: монетизація присутня, але ніколи не блокує гру. Сильний орієнтир для «ненав'язливо».
- **Повна укр. локалізація** UI → підтверджує наш мультимовний вимог; стежити за довжиною рядків.
- Кастомізація (підбірки, фільтри, grid/list) для великих бібліотек.

## ⚠️ Plarium Play (нативний клієнт, **НЕ залогінено** → видно discovery-стан) `[?]` для внутрішніх екранів

**Logged-out discovery:** «Try Our Top Games» герой (Vikings: War of Clans + CTA `Learn More` + стрічка тамбнейлів інших ігор); «Browse All Games» з фільтрами `Genre / Features / Theme`; сітка карток ігор. Ліва нав: `All Games / My Library / Notifications / Friends / Messages / Settings` + `Log in`.

**Висновки для WGC:**
- **Залогінений-out стан = вітрина-діскавері** → показує цінність ДО входу. Релевантно нашій **вторинній аудиторії (нові гравці, онбординг)** — WGC може давати контент/крос-промо ще до логіну.
- Діскавері по `Genre/Features/Theme` — модель навігації портфоліо.
- Внутрішні `My Library` / залогінені екрани не зняті (не виконано вхід).
