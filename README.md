# Wargaming Game Center — Редизайн

Редизайн десктопного лаунчера **Wargaming Game Center**: швидкий доступ до ігор
+ хаб всесвіту WG (новини, івенти, оновлення, нові ігри).

## Коротко

- **Мета**: максимально швидкий запуск ігор (time-to-play) + зменшити тертя міграції без втрати конверсії.
- **Аудиторія**: гравці 35+ всесвіту Wargaming і нові гравці.
- **Платформа**: десктоп.
- **Скоуп**: уся екосистема WG.
- **Deliverables (фаза 1)**: дослідження + UX-стратегія, wireframes ключових екранів.
- **North-star**: time-to-play + залучення в контент.
- **Пріоритетний контент**: новини/апдейти, івенти/календар, магазин/офери (ненав'язливо).
- **A11y**: читабельність 35+, WCAG AA, багатомовність.
- **Формат**: поетапно, глибоко (discovery → IA → макети → валідація).
